# richyplayer

a video player library for pygame(-ce), works on native and web

remember to add PEP 723 header to the top of your file for `pygbag`, like this:

```python
# /// script
# dependencies = [
#   'numpy',
#   'opencv-python',
#   'pygame-ce'
# ]
# ///

import richyplayer

async def main():
    # code here
    ...

asyncio.run(main())
```
